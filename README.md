# Titre du projet
TerraGauge

## Pour commencer
Notre projet (Patrick Addison, Léo Diotallevi, Maël Yvetot) pour le concours Trophées de NSI.
Thème: Nature et Informatique
Concept: Une application pour mesurer son empreinte carbone avec un classement avec ses amis.


### Prérequis
- Python 3.11+ mais pas plus de 3.12 (sinon kivy pas compatible) (3.11.5 recommandé)

### Installation
- Installer les dépendances :
```bash
pip install -r requirements.txt

```

## Démarrage
Pour démarer le projet, utiliser python pour démarer main.py

VS code, kivyMd, sqlite
